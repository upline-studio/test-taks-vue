# Test Task Vue

Задача: 
Реализовать приложение, которое будет отображать список рецептов на основе этих данных:
https://dummyjson.com/docs/recipes

Необходимо выводить рецепты в виде горизонтального списка:
- Название рецепта
- Изображение рецепта
- Список ингредиентов (первые 5 ингредиентов)

Необходимо реализовать пагинацию для отображения рецептов по 20 рецептов на странице.

Необходимо ознакомиться с выдержками из правил компании по написанию кода (RULES.md) и попробовать следовать им в своем решении.

Используйте Vue 3 Composition API SFC setup. Именуйте стили по БЭМ методологии с использованием sass (см. RULES.md).
